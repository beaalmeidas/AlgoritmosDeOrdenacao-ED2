# Algoritmos de Ordenação - ED2
Projeto desenvolvido para a disciplina de Estruturas de Dados II, contendo a implementação de diversos algoritmos de ordenação em Java, com um menu interativo para escolher qual algoritmo executar.

<br>

Algoritmos disponíveis:
- Bubble Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Selection Sort

<br>

## Como rodar
**1.** Clone o repositório:
```
git clone https://github.com/beaalmeidas/AlgoritmosDeOrdenacao-ED2.git
```

**2.** Compilar todos os arquivos:
```
javac -d bin (Get-ChildItem -Recurse -Filter *.java | % { $_.FullName })
```

**3.** Rodar a aplicação:
```
java -cp bin MainMenu
```
